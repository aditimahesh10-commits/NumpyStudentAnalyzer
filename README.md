# Student Performance Analyzer (NumPy)

This mini project analyzes marks of 50 students across 5 subjects using NumPy.

## Features
- Class average, highest, lowest marks
- Subject-wise performance analysis
- Student topper detection
- Filtering students above 80 and failing below 35
- Grade assignment (A/B/C/D)
- Grade distribution summary

## Skills Used
NumPy arrays, axis operations, boolean masking, aggregation, np.argmax, np.select
